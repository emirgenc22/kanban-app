# 🧩 Kanban Task Management App

A modern and interactive **Kanban board** for task management.  
Includes features like user authentication, task assignments, priority levels, and real-time notifications – all powered by **HTML**, **CSS**, and **JavaScript (localStorage)**.

## ✨ Features

- 🧑‍💼 User registration and login (with avatars & roles)
- 📝 Task creation, assignment, and editing
- 🟨 Task status: To Do → In Progress → Done
- 🔔 Notification system with sound alerts
- 🏢 Department filtering (Software, Marketing, etc.)
- 📅 Due dates and task priority system
- 🔒 LocalStorage-based persistence

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
- Font Awesome icons
- GitHub Pages (for deployment)
